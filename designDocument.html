Sixteen Puzzle

Description

The 15 puzzle (also called Gem Puzzle, Boss Puzzle, Game of Fifteen, Mystic Square and many others) is a sliding puzzle having 15 square tiles numbered 1â€“15 in a frame that is 4 tiles high and 4 tiles wide, leaving one unoccupied tile position. Tiles in the same row or column of the open position can be moved by sliding them horizontally or vertically, respectively. The goal of the puzzle is to place the tiles in numerical order.

Classes

  • Permutation.java
    • Generates and stores a random permutation on n items
    • isEven() - returns if the permutation is even or not
    • applyToList(List<T> someList), applyToArray(T[] someArray), applyTo2DArray(T[][] some2DArray) - applies the permutation stored in the object to the input
  • Tile.java
    • Extends javafx.scene.control.Button
    • Stores the value and position of the Tile
    • Listens for clicks and calls a Board's swap(Tile t) function
    • getValue() - returns the value stored in the tile
  • Board.java
    • Extends javafx.scene.layout.GridPane
    • Stores an array of 16 Tiles, one of which is null
    • swap(Tile t) - swaps t with the null tile if the move is valid
  • App.java
    • Top object
    • startGame() - starts the game, handling all setup
